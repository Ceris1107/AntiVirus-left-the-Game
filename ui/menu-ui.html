HTML разметка

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц.

Основные элементы:

  • Структурные теги (header, main, footer)
  • Заголовки (h1-h6)
  • Параграфы (p)
  • Списки (ul, ol, li)
  • Ссылки (a) и изображения (img)
  • Контейнеры (div, section, article)

CSS стили

CSS (Cascading Style Sheets) - это язык стилей, используемый для описания внешнего вида документа, написанного на HTML.

Возможности CSS:

  • Изменение цветов, шрифтов и размеров
  • Позиционирование элементов на странице
  • Адаптивный дизайн для разных устройств
  • Анимации и переходы
  • Создание сложных макетов (Flexbox, Grid)
  • Работа с тенями и градиентами

JavaScript

JavaScript - это язык программирования, который позволяет создавать интерактивные веб-страницы.

Основные функции:

  • Обработка действий пользователя
  • Изменение содержимого страницы
  • Работа с данными и вычисления
  • Отправка и получение данных с сервера
  • Создание анимаций и игр
  • Валидация форм
